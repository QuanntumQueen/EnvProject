# Обзор системы 
Проблема пластикового загрязнения.
Ежегодно на суше остается более 150 миллионов тонн пластиковых отходов, которые загрязняют парки, улицы, леса и городские территории. Существующие методы уборки неэффективны и требуют значительных ресурсов.

# Ключевые проблемы

1. Отсутствие систематического мониторинга загрязненных территорий
2. Ручная сортировка и уборка пластика - трудоемкий процесс
3. Неспособность оперативно оценить масштабы загрязнения
4. Отсутствие данных для планирования уборочных мероприятий

# Технологическое решение
EcoGuard - интеллектуальная система компьютерного зрения, которая автоматически обнаруживает пластиковое загрязнение на различных территориях и планирует оптимальную стратегию уборки. Система использует передовую модель YOLOv8x для точного определения типа пластиковых отходов.

# Основные возможности
Автоматическое обнаружение пластика

7 целевых классов пластиковых отходов:
1. Бутылка (bottle)
2. Стакан (cup)
3. Вилка, ложка, нож (fork, spoon, knife)
4. Посуда (bowl)
5. Пакет (bag)

Технические характеристики:

- Высокая точность детекции: модель YOLOv8x, обученная на COCO dataset
- Работа с изображениями любого качества

Анализ загрязнения территорий
- Количественная оценка пластикового мусора
- Определение наиболее загрязненных зон
- Статистика по типам пластиковых отходов
- Рекомендации по типу уборки

Интеллектуальный анализ загрязнения:
- Количественная оценка: подсчет пластиковых объектов на территории
- Качественный анализ: определение типов пластикового мусора
- Визуализация результатов: наложение bounding boxes на изображения
- Статистическая отчетность: формирование сводных данных по загрязнению

# Быстрый старт
Шаг 1: Подготовка окружения

- Установка всех необходимых компонентов
```
pip install ultralytics kaggle opencv-python matplotlib numpy
```
- Скачивание датасета пластиковых бутылок
```
kaggle datasets download -d siddharthkumarsah/plastic-bottles-image-dataset
```
- Распаковка
```
unzip -q -n plastic-bottles-image-dataset.zip -d plastic_bottles_dataset
```
